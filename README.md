# Endpoint Compliance Readiness Checker

A read-only PowerShell toolkit for basic Windows endpoint readiness checks.

## Features

- OS and build context
- Disk space summary
- Firewall profile status
- Defender status context
- BitLocker context where available
- Pending reboot indicator
- CSV, JSON, and HTML reports

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Endpoint_Compliance_Readiness_Checker.ps1
```

## Safety

Diagnostic-only. It reports readiness context for support review.
